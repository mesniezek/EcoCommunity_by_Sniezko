# EcoCommunity 🌿
**A Comprehensive System Supporting Neighborhood and Estate Communities.**

---

## 📖 About the Project
**EcoCommunity** is designed to revolutionize the way residents interact with their neighbors and estate managers. The project addresses the lack of efficient information flow, resource waste, and the difficulty of accessing local services. 

Unlike generic Facebook or WhatsApp groups, this system provides a dedicated space for both formal administration matters and informal neighborly cooperation.

### Key Problem Solved
* **Resource Waste**: Residents often buy tools (like drills) that they use rarely, while neighbors might already own them but have no way to share them.
* **Communication Gaps**: Repair statuses and administrative announcements are often outdated or lost in chat history.
* **Efficiency**: Eliminates the need for phone calls or physical meetings with administration to resolve simple issues.

---

## ✨ Key Features
* **📦 Sharing Economy Module**: A dedicated space for reserving and borrowing items from neighbors.
* **🛠 Technical Reporting System**: Report failures in common areas with a real-time repair status tracker.
* **📢 Digital Notice Board**: Official messages from the manager with priority filtering (Urgent, Info, Event).
* **📅 Duty Schedule**: An organized calendar of resident responsibilities.

## 🚀 Getting Started

### Prerequisites
* **Node.js**: v21.0 or higher
* **Expo Go**: Installed on your mobile device (iOS/Android)
* **API Access**: Ensure the .NET backend is running and accessible

### Installation & Running
1. **Clone the repository:**
 ```bash
 git clone [https://github.com/your-username/EcoCommunity.git](https://github.com/your-username/EcoCommunity.git)
 ```
2. ** Install dependencies:**
```bash
npm install
```
3. **Start the application:**
```bash
npx expo start
```
4. Launch on device:
* Physical Device: Scan the QR code using the Expo Go app (Android) or Camera app (iOS).
* iOS Simulator: Press i in the terminal.
* Android Emulator: Press a in the terminal.
